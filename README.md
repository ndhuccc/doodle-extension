# Doodle Overlay 塗鴉白板

A Chrome extension that overlays a drawing canvas on any webpage — great for teaching, presentations, and annotations.

## Features

| Tool | Shortcut | Description |
|------|----------|-------------|
| ✏️ Pen | `P` | Freehand drawing |
| 🟡 Highlight | `H` | Semi-transparent highlighter |
| ➡️ Arrow | `A` | Click and drag to draw arrows |
| 🧹 Eraser | `E` | Erase strokes |
| 🔴 Laser | `L` | Red laser pointer with 1-second fading trail |
| 📋 White Panel | `W` | Fill canvas white (whiteboard mode) |

### Other Shortcuts

| Key | Action |
|-----|--------|
| `Alt+Shift+D` | Toggle doodle mode on/off |
| `Ctrl+Z` | Undo last stroke |
| `C` | Clear all drawings |

### Additional Features

- **10-color palette** — red, orange, yellow, green, blue, purple, pink, teal, white, black
- **Adjustable brush size** — vertical slider in toolbar
- **Draggable toolbar** — drag the ⠿ handle to reposition
- **Scroll-following canvas** — drawings stay anchored to page content as you scroll
- **Touch support** — works on touchscreen devices

## Installation

1. Clone or download this repo
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select this folder

## Usage

- Click the ✏️ FAB button (bottom-right) or press `Alt+Shift+D` to activate
- Select a tool from the toolbar or use keyboard shortcuts
- Press `L` for laser pointer mode during presentations
- Press `W` for full whiteboard mode
